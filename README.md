# Zephyr Lash Art Studio Web

![React](https://img.shields.io/badge/React-20232A?logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript)
![Vite](https://img.shields.io/badge/Vite-646CFF?logo=vite)

Frontend web application for a modern lash and brow studio, built with **React + TypeScript**.  
This project serves as a scalable client-facing foundation for appointment booking, service promotion, and future business growth.

## Overview

Zephyr Lash Art Studio Web is designed to provide a premium digital experience for beauty clients seeking services such as:

- Lash Extensions
- Brow Lamination
- Lash Lift & Tint
- Personalized Beauty Consultations

The website focuses on clean design, smooth navigation, and a booking-first experience.

## Current Features

- Responsive landing page
- Premium hero section
- Services showcase
- Gallery / portfolio section
- Booking interface with date & time selection
- Testimonials section
- Contact section with social links
- Mobile-friendly layout

## Planned Features

- Real-time booking system
- Backend API integration
- Admin dashboard
- Appointment management
- Calendar sync (Google Calendar / custom scheduler)
- Automated reminders
- Client database
- Payment gateway integration

## Getting Started

Install dependencies:

npm install

Run locally:

npm run dev

Build for production:

npm run build

## Notes

This repository currently contains the frontend prototype / client experience layer only.

No production credentials, private client data, or backend business logic are included.
